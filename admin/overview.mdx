The admin panel is a local-only tool that runs while the development server is active. It is not deployed to GitHub Pages and is not accessible to the public — it exists solely to help you manage sketch data from your own machine. Open it by navigating to [http://localhost:3000/admin](http://localhost:3000/admin) while `npm start` is running.

<Note>
  The admin panel has no password protection. It is intended for local use only and is never exposed in the deployed site.
</Note>

## Two tabs, two tools

The panel is divided into two tabs accessible from the navigation bar at the top:

- **Place Sketches** — displays each unlocated sketch image one at a time alongside an interactive Mapbox map. Click the map to pin a sketch to its real-world location, then confirm or skip before moving on.
- **Upload Images** — provides drag-and-drop zones for adding new sketch or type specimen images directly from the browser. The server processes each file using the appropriate Python script.
